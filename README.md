# 개요

- AI 음성 인식 기반 헬스장 출입 시스템을 위한 백엔드 서버

# 핵심 구현 내용

- 회원 등록
- 음성인식 기반 회원 로그인(AI모델 연동)
- 회원 조회, 삭제, 수정 등 기본적인 CRUD 지원
- 로그인 시 데이터베이스에 출입 로그 기록
